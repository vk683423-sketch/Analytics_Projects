# Project Overview 
* This Project is a sales & business proformance dashboard built using power BI.
* The dashboard provides insight into sales revenue profit, and customers performance accorse different regions, and states.
* It hepls business tracks sales trends , regional proformance and payments patterns to make better data driven decisions.

# Objective
The main objective of this dashboard are:
* Analyze total revenue profit and profit margin.
* Track customers-wise sales performace.
* Identify Top performaning states
* Compare revenue by region
* understand payment mode distribution.
* Monitor monthly sales trend.
